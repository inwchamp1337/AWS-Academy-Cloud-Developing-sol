
# AWS Academy Cloud Developing - Solutions Overview

## Overview

This repository documents my learning journey through the AWS Academy Cloud Developing course. By working through each lab, I gained hands-on experience with AWS services such as DynamoDB, Lambda, and API Gateway. Completing these labs not only deepened my understanding of cloud development but also helped me build practical skills in Python and AWS automation.

## Lab Summaries

### Lab 3.1: Getting Started with AWS CLI & Python
- Script: `ultimate_run.py`
- Content: Introduction to AWS CLI, credential setup, and basic Python scripting to connect with AWS.
- See details and usage in [Lab 3.1/readme.md](Lab 3.1/readme.md)

### Lab 5.1: DynamoDB Automation
- Script: `lab51_one_shot_pass.py`
- Content: Automates DynamoDB table creation, loads data from JSON, creates a GSI, and scans data.
- Guide and troubleshooting in [Lab 5.1/readme.md](Lab 5.1/readme.md)

### Lab 6.1: Building a REST API with API Gateway
- No automation script (manual steps required via AWS Console)
- Content: Create REST API, Mock Integration, Mapping Template, and deploy API for the website.
- Step-by-step guide in [Lab 6.1/readme.md](Lab 6.1/readme.md)
